Get started

How does this work?

1. Install MetaMask Browser Plugin

2. Register at MetaMask Plugin

3. Inside Metamask select on the top left corner "Ropsten Test Network".


4. Get Test Ether by clicking on "Buy" inside MetaMask and than on "Ropsten Test Faucet".
On the website select "request 1 ether from faucet"



3. Choose your Artwork (any digital file!) and generate a unique ID (hash)
(What is a hash?)

Drag and Drop your File in the Box




3. The Smart Contract address to use is:
0x7427c2e4b65b5d01e253fc1e090b6a17d9c5b7e5
Smart Contract on Blockexplorer

4. Send an Ethereum transaction to the Smart Contract - including your hash.
The Hash is used to calculate your transaction data. Copy this data into the "transaction data (optional)" field of metamask.
Leave the "Amount" field blank.




How to confirm your Art?

1. Go to the Smart Contract on the Blockexplorer and click on the tab "Transaction".

2. Compare your wallet adress with the receiving addresses. Click on the matching transaction hash.

3. On the bottom of the transaction's page you'll find the Input Data. If you see the SHA-256 hash value of your artwork - your art is on the blockchain.


Why register Art on the Blockchain?

Ethereum is a decentralized global network of computers that run the Ethereum client software. This client software creates a continues blockchain in packing transactions into blocks. The blocks are timestamped and tamperproof.
So if you send a transaction to the ethereum network including a hash value of your artwork, it is proven, that:
You owned the digital file needed to generate the hash, at the moment the transaction occured.
